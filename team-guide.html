Панель управления
Как зайти в панель Aeza

Вход в панель

  1. Откройте aeza.net
  2. Нажмите «Войти» в правом верхнем углу
  3. Введите email и пароль, которые использовали при регистрации
  4. После входа перейдите в раздел «Виртуальные серверы»
💡
В панели вы увидите свой сервер, его IP-адрес, статус и срок аренды. IP-адрес нужен для подключения через терминал.

Что можно делать в панели

  • Перезагрузить сервер — кнопка «Перезагрузить» в карточке сервера
  • Посмотреть IP-адрес и пароль — в карточке сервера (глазик рядом с паролем)
  • Продлить аренду — кнопка «Продлить»
  • Посмотреть баланс — в правом верхнем углу
02
Подключение
Как зайти на сервер через терминал

Открываем терминал

  • Windows 10/11 — нажмите Win + R, введите cmd и нажмите Enter
  • macOS — откройте приложение Терминал (через Spotlight: Cmd + Пробел → «Terminal»)
  • Linux — Ctrl + Alt + T

Подключаемся к серверу

Введите команду, заменив ВАШ_IP на IP-адрес из панели Aeza:

ssh root@ВАШ_IP
  1. При первом подключении спросит «Are you sure you want to continue connecting?» — введите yes
  2. Введите пароль из панели Aeza
⚠️
Пароль при вводе не отображается — это нормально! Просто печатайте и нажмите Enter. В Windows вставка — правой кнопкой мыши (не Ctrl+V).
✅
Если видите root@имя-сервера:~# — вы успешно подключились!
03
Перезапуск
Как перезапустить бота

Способ 1: Мягкий перезапуск (рекомендуемый)

Подключитесь к серверу (шаг 02) и выполните:

openclaw gateway restart

Дождитесь сообщения об успешном перезапуске. Бот заработает в течение 10-30 секунд.

Способ 2: Остановка и запуск отдельно

Если мягкий перезапуск не помог:

openclaw gateway stop
openclaw gateway start

Способ 3: Перезагрузка сервера целиком

Крайний случай — если ничего не помогает:

reboot
⚠️
После перезагрузки сервера нужно подождать 1-2 минуты и подключиться заново. OpenClaw запустится автоматически.

Способ 4: Перезагрузка через панель Aeza (без терминала)

Если не можете зайти на сервер через терминал:

  1. Зайдите в панель Aeza
  2. Откройте карточку сервера
  3. Нажмите кнопку «Перезагрузить»
  4. Подождите 1-2 минуты — бот запустится сам
04
Проверка
Как проверить что всё работает

Быстрая проверка

Подключитесь к серверу и выполните:

openclaw gateway status

Если видите running — шлюз работает.

Полная диагностика

openclaw doctor

Команда проверит всё: конфигурацию, подключение к AI, каналы. Если что-то не так — подскажет, что исправить.

Просмотр логов (что делает бот прямо сейчас)

openclaw logs --follow
💡
Чтобы выйти из просмотра логов, нажмите Ctrl + C
05
Сервер
Как продлить сервер на Aeza

Пошагово

  1. Зайдите на aeza.net и войдите в аккаунт
  2. Перейдите в «Виртуальные серверы»
  3. Нажмите на сервер → увидите дату окончания аренды
  4. Нажмите «Продлить»
  5. Выберите срок продления (месяц / несколько месяцев)
  6. Оплатите
🔴
Не допускайте истечения аренды! Если сервер не продлить вовремя, Aeza может удалить его через несколько дней. Все данные и настройки бота будут потеряны. Настройте автопродление или поставьте напоминание за 3 дня до окончания.

Как пополнить баланс Aeza

  • Банковская карта — российские карты работают
  • СБП — быстрый перевод
  • Криптовалюта — Bitcoin, USDT и др.
💡
Стоимость сервера — от ~900 ₽/мес (тариф DEs-2: 2 vCPU / 4 ГБ RAM). Включите автопродление в настройках сервера!
06
Подписка
Как продлить подписку Claude

Вариант 1: Через Авито (самый простой и дешёвый)

  1. Перейдите к проверенному продавцу на Авито
  2. Выберите нужный тариф: Claude Pro ($20/мес) или Claude Max ($100/мес)
  3. Оплатите через Авито
  4. Продавец продлит подписку и пришлёт подтверждение
💰
Через Авито выходит дешевле, чем напрямую иностранной картой. Плюс не нужна иностранная карта.

Вариант 2: Напрямую на claude.ai

  1. Зайдите на claude.ai
  2. Войдите в аккаунт
  3. Перейдите в Settings → Subscription
  4. Оплатите иностранной картой (Visa/Mastercard зарубежного банка)

После продления: обновляем токен на сервере

Если после продления подписки бот перестал работать, возможно нужно обновить setup-token:

  1. На своём компьютере установите Node.js (если ещё не установлен): скачайте с nodejs.org (кнопка LTS) → Next → Next → Finish
  2. Откройте терминал на компьютере и установите утилиту Claude: 
    npm install -g @anthropic-ai/claude-code
  3. Получите новый токен: 
    claude setup-token
    Откроется браузер — войдите в аккаунт Claude и подтвердите доступ.
  4. Скопируйте новый токен
  5. Подключитесь к серверу и запустите: 
    openclaw onboard
  6. Вставьте новый токен когда мастер попросит
  7. Перезапустите: openclaw gateway restart
💡
Обычно при продлении подписки токен не меняется и бот продолжает работать. Обновлять токен нужно только если бот перестал отвечать после продления.
🔴
Не забывайте продлевать! Без активной подписки Claude бот перестанет отвечать. Продлевайте заранее — за 1-2 дня до окончания.
07
Обновление
Как обновить OpenClaw

Обновление до последней версии

Подключитесь к серверу и выполните:

npm update -g openclaw

После обновления перезапустите бота:

openclaw gateway restart
💡
Обновлять рекомендуется раз в 1-2 недели. Разработчики часто выпускают улучшения и исправления.

Проверить текущую версию

openclaw --version
08
Проблемы
Что делать если бот не отвечает

Чек-лист по порядку

  1. Напишите боту ещё раз — иногда сообщение просто теряется
  2. Подождите 2-3 минуты — бот может думать над сложным запросом
  3. Проверьте статус сервера в Aeza — должен быть «Активен»
  4. Зайдите на сервер и выполните openclaw gateway status
  5. Если статус не running — запустите: openclaw gateway start
  6. Если running но не отвечает — перезапустите: openclaw gateway restart
  7. Если ничего не помогает — запустите openclaw doctor

Частые причины

  • Закончилась подписка Claude → продлите (шаг 06)
  • Закончилась аренда сервера → продлите (шаг 05)
  • Сервер «упал» → перезагрузите через панель Aeza
  • Исчерпан лимит запросов → подождите, лимит обновится

Не получается зайти на сервер через терминал

  • «Connection refused» — сервер выключен. Включите через панель Aeza
  • «Connection timed out» — проверьте IP-адрес и включите VPN
  • «Permission denied» — неправильный пароль. Посмотрите актуальный в панели Aeza
